这个仓库是用来备份知乎上被删除的关于信仰的回答，以及保证我自己文章上所引用的链接有效，原因如下：

1\. 知乎上有关信仰的回答大概由于zz原因经常被删除。

2\. 其他人的仓库目录变动，引用失效。

被删除的回答见本项目中的两个 .docx 文件，里面有原文、对应回答下答主回复的评论。

文章：

[信仰、爱和原谅](https://zhuanlan.zhihu.com/p/592553117)

[人生的意义](https://zhuanlan.zhihu.com/p/637607393)
